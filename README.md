🧠 Sobre o Projeto

Este projeto foi desenvolvido para facilitar o controle e a reserva de instrumentos musicais em ambientes como:

Escolas de música
Igrejas
Estúdios
Bandas

Com ele, é possível evitar conflitos de uso, organizar horários e manter um histórico de reservas.

✨ Funcionalidades
📅 Cadastro de reservas por data e horário
🎹 Listagem de instrumentos disponíveis
🔒 Controle de conflitos (evita reservas duplicadas)
👤 Autenticação de usuários (opcional)
📊 Histórico de reservas
❌ Cancelamento de reservas
